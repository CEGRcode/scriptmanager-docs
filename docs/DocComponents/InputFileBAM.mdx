### File input (BAM)

This script processes [BAM-type files][bam-format] so make sure your input is properly formatted and uses the appropriate `.bam` extension if you are working from the GUI.

:::caution
Make sure your BAM input files are [sorted][sort-bam] and [indexed][bam-indexer].
:::
